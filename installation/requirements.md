# Requirements

## Java

| MC Version      | Java Version |
| --------------- | ------------ |
| 1.17 -> 1.20.6  | Java 17      |
| 1.20.6 or above | Java 21      |

## Server Specs

### Minimum

* RAM: 2GB
* CPU: Anything decent
* Storage: 5GB
* A side of pancakes

### Recommended

* RAM: 4GB
* CPU: Decent enough
* Storage: 20GB
* A side of Pancakes
